# Scripture Language Academy

A self-paced React web app for studying Biblical Hebrew and Aramaic.

## What is included
- Biblical Hebrew Module 1
- Aramaic Module 1
- Interactive lesson selection
- Progress tracking
- Auto-save in the browser using local storage
- Notes area
- Review questions

## How to upload to GitHub
1. Download this folder or ZIP file.
2. On GitHub, click **New repository**.
3. Name it `language-academy-app`.
4. Click **Add file** > **Upload files**.
5. Upload everything from this project.
6. Click **Commit changes**.

## How to deploy on Vercel
1. Sign in to Vercel.
2. Click **Add New** > **Project**.
3. Import your `language-academy-app` repository from GitHub.
4. Leave the default settings as they are.
5. Click **Deploy**.

Vercel should detect this as a Vite React app automatically.
